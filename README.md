# Data Science Learning Portfolio

**Автор:** Вожегов Алексей

Этот репозиторий — коллекция моих учебных и исследовательских проектов по машинному обучению. Здесь собраны реализации ключевых задач ML на Python с акцентом на чистоту кода, визуализацию и интерпретацию результатов.

## Структура

- **`src/`** — исходный код с реализациями моделей
- **`plots/`** — сохранённые графики и визуализации
- **`other/`** — вспомогательные утилиты и данные

## Основные разделы

| Раздел | Описание | Ключевые библиотеки |
| :--- | :--- | :--- |
| **Классификация** (`classification.py`) | Решение задач бинарной и многоклассовой классификации. Сравнение моделей (Logistic Regression, Random Forest, SVM), оценка метрик (Accuracy, Precision, Recall, F1, ROC-AUC), подбор гиперпараметров. | `scikit-learn`, `pandas`, `matplotlib`, `seaborn` |
| **Регрессия** (`regression.py`) | Построение и оценка моделей регрессии (Linear, Ridge, Lasso, Decision Tree). Анализ остатков, визуализация предсказаний. | `scikit-learn`, `statsmodels`, `matplotlib` |
| **Кластеризация** (`clustering.py`) | Применение алгоритмов K-Means, иерархической кластеризации для сегментации данных. Методы выбора числа кластеров (локтевой метод, силуэт). | `scikit-learn`, `scipy`, `matplotlib`, `seaborn` |
| **Ассоциативные правила** (`association.py`) | Поиск ассоциативных правил (например, для анализа корзины покупок) с помощью алгоритма Apriori. | `mlxtend`, `pandas` |
| **Временные ряды** (`time_series.py`) | Базовый анализ временных рядов: декомпозиция, скользящие средние, экспоненциальное сглаживание, оценка стационарности. | `pandas`, `statsmodels`, `matplotlib` |

## Используемые технологии

- **Python** 3.9+
- `numpy`, `pandas` — обработка данных
- `matplotlib`, `seaborn`, `plotly` — визуализация
- `scikit-learn` — классические ML-модели и метрики
- `statsmodels`, `scipy` — статистический анализ
- `mlxtend` — поиск ассоциативных правил
